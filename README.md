# Fraud-Detection-System-using-Machine-Learning

## Project Overview -

This project focuses on building a machine learning model to detect fraudulent transactions using historical transaction data. 
The goal is to identify suspicious activities in real-time and minimize financial losses.

## Key Features -
- Data preprocessing and feature engineering
- Time-based train-test split to avoid data leakage
- Handling imbalanced data using SMOTE
- Model training using Logistic Regression & XGBoost
- Performance evaluation using ROC-AUC, Precision, Recall, and F1-score
- Threshold tuning for better fraud detection
- Model persistence using .pkl files

## Technologies Used -

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* Matplotlib
* Imbalanced-learn (SMOTE)
* Joblib

## Model Evaluation Metrics -

- ROC-AUC Score
- Confusion Matrix
- Precision & Recall
- F1 Score
- Precision-Recall Curve

## Future Improvements -

- Hyperparameter tuning
- Real-time fraud detection API (Flask/FastAPI)
- Deployment on cloud (AWS/GCP)
- Advanced feature engineering
